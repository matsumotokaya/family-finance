# 松本家 家計ダッシュボード

家族全員が「お金の現実」を直視して、支出を見直すためのWebアプリです。

---

## なぜ作ったか

収支の全体像が見えないと、危機感が生まれません。
このアプリは「今月いくら使いすぎているか」「このままでは2028年以降どうなるか」を、家族全員がスマホで確認できるようにするために作りました。

---

## 主な機能

### 1. 月次収支の一目表示
- 支出・見込み収支を大きな数字で表示
- **赤字月はヘッダーと画面が赤くなり、警告バナーが出ます**
- 取引明細をタップで展開（デフォルトは折りたたみ）

### 2. 貯蓄目標の達成度
- 月次の貯蓄目標（現在：¥100,000/月）に対してどれだけ貯蓄できているかをバーで表示
- 不足額も明示

### 3. 住宅ローン減税カウントダウン
- 2028年12月にローン減税が終了するまでの残り月数を常に表示
- **終了後は毎月¥115,410の追加負担**になることを常に意識できるようにしています

### 4. 口座残高の推移
- 三菱UFJ・りそなの残高推移を折れ線グラフで表示

### 5. カード請求額の推移
- VIEWカード・ルミネカードの月次請求額を折れ線グラフで表示
- **上限¥40,000のラインを赤い破線で表示**（VIEWカード未来の予算上限）
- 2025年12月〜の推移を確認できる

### 6. カード利用明細
- VIEWカード（未来）・VIEWカード（夏弥）・ルミネカード（夏弥）をタブで切り替え
- 選択中の月の全明細を表示

### 7. VIEWカード超過アラート
- 未来のVIEWカードが上限¥40,000を超えた場合に警告バナーを表示

## データ更新ルール（月次運用）

毎月1回、以下の手順でデータを更新します。

### ステップ1：銀行 → スクリーンショットで送る

| 銀行 | アプリ操作 | 送り方 |
|------|-----------|-------|
| 三菱UFJ銀行 | アプリ「入出金明細」→ 当月全件表示 | スクリーンショット複数枚 |
| りそな銀行 | アプリ「普通預金」→ 当月表示 | スクリーンショット複数枚 |

**スクリーンショットのルール:**
- 日付・金額・残高・取引先名がすべて写っていること
- 月頭〜月末の全取引が途切れなく収まるよう複数枚撮る
- 「月途中」のデータでもOK（ダッシュボードに「データ不完全」と表示される）

### ステップ2：カード → CSVで送る

| カード | ダウンロード場所 | ファイル形式 |
|--------|----------------|------------|
| VIEWカード（夏弥・未来） | [VIEW's NET](https://www.viewnext.ne.jp/) → 「利用明細照会」→ CSV出力 | `.csv`（Shift-JIS） |
| ルミネカード（夏弥） | 同上（同じVIEW's NETで管理） | `.csv`（Shift-JIS） |
| 楽天カード | 楽天e-NAVI → 「ご利用明細」→ CSV出力 | `.csv` |
| 高島屋カード | 高島屋カード会員サービス → 明細ダウンロード | `.csv` or `.xlsx` |

**CSVファイルの置き場:**
```
family-budget/credit-card/
  ご利用明細_YYYYMM.csv       ← VIEWカード（夏弥・未来混在、番号末尾で自動判別）
  ご利用明細_YYYYMM (1).csv  ← ルミネカード（夏弥）
```
ファイル名はダウンロード時のままでOK。同じフォルダに入れてClaudeに「カード更新して」と伝えるだけ。

### ステップ3：Claudeへの依頼文（コピペ用）

```
今月分のデータを更新してください。
・三菱UFJ・りそなのスクリーンショット：[添付]
・カードCSV：credit-card/ フォルダに追加済み
対象月: YYYY年MM月
```

### ステップ4：デプロイ

Claudeがデータを更新したら、以下をターミナルで実行：

```bash
vercel --prod
```

---

### 更新タイミングの目安

| タイミング | 対象データ |
|-----------|-----------|
| 毎月5日前後 | カードCSV（前月の請求確定後） |
| 毎月末 | 銀行スクリーンショット（月末残高確認） |
| 随時 | 銀行スクリーンショット（月途中確認したいとき） |

[data/config.json](data/config.json) を編集してください。

```json
{
  "familyName": "松本家",
  "monthlyTargetSavings": 100000,   ← 月次貯蓄目標（円）

  "mortgageEndMonth": 12            ← 住宅ローン減税 終了月
}
```

## 個人ページ（松本未来）

ハンバーガーメニューから「👤 松本未来」でアクセス。

### 機能
- 月ごとの収支を表示（12月〜3月、デフォルトは最新月）
- 収入 ¥40,000 に対する支出の可視化（プログレスバー・超過時赤表示）
- **銀行引き落とし**と**VIEWカード（未来）利用明細**を一覧表示
- 各明細に「除外」ボタン → 自分の支出でない項目を合計から外せる
- 各明細に「コメント（メモ）」機能 → 吹き出しアイコンから追加

### 状態の永続化（Supabase）

除外・コメントの状態は Supabase に保存されます。ページリロード後・別デバイスからでも状態が維持されます。

| テーブル | 用途 |
|---------|------|
| `familybudget_exclusions` | 除外した明細の記録 |
| `familybudget_comments` | 明細へのメモ |

両テーブルは `person` 列で人物を区別しており、夏弥など他の家族メンバーのページを追加する際も同じテーブルを使い回せます。

**接続先**: `https://rgqduwojvylkulhyodqg.supabase.co`
**環境変数**: `.env.local` に `NEXT_PUBLIC_SUPABASE_URL` と `NEXT_PUBLIC_SUPABASE_ANON_KEY` を設定（要ローカル設定、Vercelにも登録必要）

## 未確定決済情報ページ

- 3月分・4月分を月別に切り替えて表示

- 「来月の予算」で、基準予算 ¥40,000 から今回の未確定利用分を差し引いて表示
